# net:ftp-get

FTP command to retrieve a remote file.

### Property Summary

| Property | Description |
| -------- | ----------- |
| [file](#propertyfile) | A local file to copy to. | 
| [output](#propertyoutput) | A local output to copy to. | 
| [remote](#propertyremote) | The name of the remote file. |
